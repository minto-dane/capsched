# Formal 0132: P5A-R3 E1 Source, Locking, and Lifetime Evidence Plan

Date: 2026-07-15

Status: pre-source plan model. A passing safe model and every expected unsafe
counterexample authorize only the exact default-off R3-E2 two-file disposable
layout draft.

The model walks an online, admission, publication, work, offline, retirement,
callback-drain, RCU, and free sequence. `StateSafety` requires projection
capacity before a contribution, initialized-before-accepting state, live work
ownership for pending/running callbacks, no admission after offline, preserved
active membership for nonzero contributions, and full callback/ref/RCU drain
before free.

`PlanContract` fixes the exact primary, non-parent rejected line, `B_max=64`,
single outer layer, sparse projection map, 64KiB per-rq private memory bound,
zero ordinary hot-structure growth, exact default-off E2 scope, cross-
architecture layout evidence, rq-to-one-membership locking, unbound work and
coalesced ownership, hotplug behavior, drain/RCU ordering, complete E3 race
families, immutable E4 rejection limits, and explicit cross-path/production
non-claims.

Thirty-six unsafe fault values remove one required condition each and must
produce an expected `Safety` counterexample.
