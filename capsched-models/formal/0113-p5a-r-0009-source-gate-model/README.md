# P5A-R 0009 Source Gate Model

This model checks the claim boundary for Linux patch `0009`.

It is not a runtime proof for deny-and-repick behavior. It only allows the
source gate to pass when the Linux draft, patch queue entry, source checker,
ordinary-CFS scope, dormant static key, bounded attempt-local carrier,
pre-settle picker shape, cross-path exclusion predicate, and no-ABI/no-monitor
constraints are all recorded while acceptance and runtime/protection/cost
claims remain false.
